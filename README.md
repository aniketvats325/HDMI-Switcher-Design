# HDMI-Switcher-Design

This project presents the design and implementation of a 4×1 HDMI switcher using the TMDS461 HDMI switch IC. The system is designed to connect four HDMI input sources to a single HDMI output display, allowing easy switching between different multimedia devices. 
This repository consists following:

- **Board Schematic**: Root Schematic of the HDMI Switcher.
- **PCB Routing**: The connections and pathways for signals and power on the PCB.
- **3D Model**: 3D Model of the design after routing and placement of all components.

### Tools Used

The project was developed using an open-source EDA software,** KiCad version KiCad 9.0.7**

#### The footprints of all Resistors, Capacitors and Diodes:
`0603_1608Metric_Pad1.08x0.95mm_HandSolder`
